# Plain Language

Use these translations for non-technical participants.

- Docker image: "the packaged version of your whole app"
- Registry: "the place where the final app package is uploaded for judges"
- SQLite: "the app's saved-data file"
- Backend: "the part that saves data and answers requests"
- Frontend: "the screens people click on"
- Build: "package the app so it can run"
- Push: "upload the saved code or image"
- Health check: "a quick test that the app has started correctly"

Prefer:

```text
Your app is ready to open here: http://localhost:9080
```

Avoid:

```text
The containerized runtime has bound a host port and mounted services successfully.
```
